# TP4 — Prolog · Árbol de Derivación · Filmina 11

**Materia:** Técnicas de Procesamiento del Habla  
**Bloque:** Lenguajes y Autómatas - Gramáticas  
**Unidad:** 1

---

Trabajo Práctico N°4 correspondiente a la Filmina 11. Consiste en el análisis sintáctico de 10 oraciones en español mediante árboles de derivación, construidos a partir de un banco de nodos predefinido e implementados en Prolog.

---

##  Nodos utilizados

Los árboles se construyeron a partir del siguiente **banco de nodos**:

| Símbolo | Significado |
|---------|-------------|
| `O` | Oración |
| `SN` | Sintagma Nominal |
| `SV` | Sintagma Verbal |
| `SAdj` | Sintagma Adjetival |
| `SAdv` | Sintagma Adverbial |
| `SP` | Sintagma Preposicional |
| `Pro` | Pronombre |
| `CONJ` | Conjunción |
| `Adj` | Adjetivo |
| `Adv` | Adverbio |
| `Det` | Determinante |
| `N` | Nombre |
| `P` | Preposición |
| `V` | Verbo |
